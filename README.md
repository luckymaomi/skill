这是一个开发前使用的项目启动 Skill，用来在新项目开始前完成需求澄清、项目骨架、工程规则、文档边界和生产级验收约束。

## 怎么用

新项目开始前，先让 Codex 阅读：

```text
skill.md
```

然后按本仓库里的模板创建目标项目骨架。
